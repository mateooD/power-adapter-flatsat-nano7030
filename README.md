# power-adapter-flatsat-nano7030
Placa de distribución y protección de alimentación para el entorno FlatSat híbrido, desarrollada en el Instituto Gulich en colaboración con el proyecto CubeSat Nano 70/30 de la FCEFyN – UNC, en el marco de las Prácticas Supervisadas (PS), diseñada para alimentar y validar los subsistemas de COMMS y OBC durante tareas de integración y ensayo.
